What is Docker?

Docker is a platform for developers and sysadmins to develop, deploy, and run applications with containers. The use of Linux containers to deploy applications is called containerization. Containers are not new, but their use for easily deploying applications is.

From Docker Docs https://docs.docker.com/get-started/

Install docker

  1. Install packages needed with command $ sudo apt-get install \ apt-transport-https \ ca-certificates \ curl \ gnupg-agent \ software-properties-common
  2. Add key sudo curl -fsSL https://download.docker.com/linux/ubuntu/gpg | sudo apt-key add -
  3. Add repository sudo add-apt-repository \ "deb [arch=amd64] https://download.docker.com/linux/ubuntu \ $(lsb_release -cs) \ stable"
  4. Update package index sudo apt-get update
  5. Install latest Docker CE sudo apt-get install docker-ce docker-ce-cli containerd.io
  6. Verify docker install sudo docker run hello-world

Set up a MariaDB

  • search on docker hub sudo docker search mariadb
  • Default MariaDB imagesudo docker pull mariadb/server:10.3
  • Start the docker image on port 80sudo docker run -d -p 80:80 mariadb/server:10.3
  • check image installedsudo docker images

Basic commads to work with Docker

  • List current images pulled sudo docker images
  • List running containers sudo docker ps
  • List all containers sudo docker ps -a
  • Start a container sudo docker start < name of container >
  • Stop a container sudo docker stop < name of container >
  • Restart a container sudo docker restart < name of container >
